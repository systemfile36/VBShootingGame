=====================================

    VB.Net Shooting Game By DH.IM    

=====================================

본 소프트웨어는 VB.Net을 활용하여 만든

슈팅 게임입니다.  


테스트 환경 : Microsoft Windows 8.1 64bit, x64 기반 프로세서


조작키

W : Up, Player 기체를 상단으로 이동

A : Left, Player 기체를 좌측으로 이동

S : Down, Player 기체를 하단으로 이동

D : Right, Player 기체를 우측으로 이동

Space : Shoot, 탄을 발사합니다. 

BackSpace : 일시정지 메뉴

C : Game Over, 입력 시점에 게임 중단 후 게임 오버 화면으로



게임 진행

Game Start 버튼을 누르면 기체를 선택할 수 있습니다.

선택한 기체를 조종하여 적의 탄을 피하며 적을 파괴하십시오.

시간이 지나면 난이도가 올라갑니다.

난이도가 오를 수록 적의 소환 간격이 짧아지고,

탄의 연사 속도가 빨라집니다.

시간이 지날수록, 적을 파괴할수록 점수가 상승합니다.

일정 난이도에 도달하면 보스가 등장합니다.

보스는 더 어려운 대신 더 큰 점수를 줍니다.

더욱 높은 점수를 목표로 삼으십시오.

최대 연사 속도는 정해져 있습니다. (기본값 : 0.4)

=====================================

부가 사항

프로그래밍 능력 향상을 위해 작성된 코드이기에, 

주석이 불필요하게 자세할 수도 있습니다.

가능한 한 객체지향을 준수하려 했으나 많이 부족할 수 있습니다.

간단한 코드 설명

GameObject : 게임에 등장하는 오브젝트들의 기본 틀

Form1 : 게임이 실제로 작동하는 폼

StartUp : 타이틀 화면 폼

GameInfo : 이 파일을 출력하는 창

GameOver : 게임 결과를 출력하는 폼

그 외 GameLogicMemo.txt 참조
